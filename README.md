# SyncWave

SyncWave is a distributed, asynchronous playlist migration system that automatically synchronizes YouTube playlists with Spotify using a Chrome extension client, a Next.js backend, and AWS SQS for scalable batch processing.

## Tech Stack

- Next.js (API routes, backend orchestration)
- Chrome Extension (YouTube playlist client)
- AWS SQS (asynchronous job queue)
- AWS S3 (artifacts and logs, optional)
- DynamoDB (job state and caching)
- Spotify Web API
